# Подсказка по Git

Создание репозитория:
```sh
git init
```

Добавление файлов:
```sh
git add
```

Создание коммитов:
```sh
git commit - m "Message"
```

Посмотреть историю изменений развернуто:
```sh
git log
```

Посмотреть историю изменений кратко:
```sh
git log --oneline
```

Переключение на другую ветку:
```sh
git checkout
```
